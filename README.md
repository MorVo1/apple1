# A simple Apple I emulator made in C
## Download 
```bash
$ cd
$ git clone https://github.com/morvo1/apple1
$ cd apple1
```

## Build
```bash
$ mkdir build
$ cd build
$ cmake ..
$ make -j$(nproc)
```

## Run
```bash
$ ./apple1
```
